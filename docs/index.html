What is Pokehell?

Pokehell is a mod to Friday Night Funkin', a rhythm game made by ninjamuffin99 (programmer), PhantomArcade (animator), evilsk8r (artist), and Kawaisprite (musician), originally for Ludum Dare 47.

Pokehell runs on multiple engines for Friday Night Funkin. The main engine being Psych Engine, the others being Forever Engine and Kade Engine. This is based on preference, some people don't like Psych Engine, some do, so I ported the mod to multiple engines. The additional engines will be complete when 1.3.1 rolls out.
